# Kernel Benchmark

This repository contains the kernel benchmark evaluation pipeline, including:

- `harness`: KernelBench-based compile, correctness, profiling, and timing harness.
- `optimization_loop`: multi-round agent optimization runner and prompt templates.
- `runs`: generated benchmark runs, traces, debug eval logs, and agent outputs.
- `README_compile_profile_timing.md`: notes for compile/profile timing instrumentation.

`gpucore` artifacts are intentionally excluded from git.
